Facade Pattern

חזרה לדף הבית

מטרת התבנית

תבנית Facade מספקת ממשק פשוט למערכת מורכבת, ומסתירה את המורכבות שלה מהלקוח.

מתי להשתמש?

  • כאשר רוצים לספק ממשק פשוט למערכת מורכבת
  • כאשר רוצים להסתיר את המורכבות של המערכת מהלקוח
  • כאשר רוצים להפחית את התלות בין הלקוח למערכת
  • כאשר רוצים לספק נקודת כניסה אחת למערכת

דוגמה בסיסית 

class Subsystem1:
    def operation1(self) -> str:
        return "Subsystem1: Ready!"

    def operation_n(self) -> str:
        return "Subsystem1: Go!"

class Subsystem2:
    def operation1(self) -> str:
        return "Subsystem2: Get ready!"

    def operation_z(self) -> str:
        return "Subsystem2: Fire!"

class Facade:
    def __init__(self, subsystem1: Subsystem1, subsystem2: Subsystem2):
        self._subsystem1 = subsystem1
        self._subsystem2 = subsystem2

    def operation(self) -> str:
        results = []
        results.append("Facade initializes subsystems:")
        results.append(self._subsystem1.operation1())
        results.append(self._subsystem2.operation1())
        results.append("Facade orders subsystems to perform the action:")
        results.append(self._subsystem1.operation_n())
        results.append(self._subsystem2.operation_z())
        return "\n".join(results)

# שימוש
def client_code(facade: Facade) -> None:
    print(facade.operation())

if __name__ == "__main__":
    subsystem1 = Subsystem1()
    subsystem2 = Subsystem2()
    facade = Facade(subsystem1, subsystem2)
    client_code(facade)

יתרונות

  • מספקת ממשק פשוט למערכת מורכבת
  • מסתירה את המורכבות של המערכת מהלקוח
  • מפחיתה את התלות בין הלקוח למערכת
  • מספקת נקודת כניסה אחת למערכת

חסרונות

  • עלולה ליצור קוד מורכב
  • עלולה ליצור קוד קשה לתחזוקה
  • עלולה ליצור קוד קשה להבנה
  • עלולה ליצור קוד קשה לבדיקה

השוואה לתבניות אחרות

תבנית הבדל
Adapter מתמקדת בהתאמת ממשקים, בעוד Facade מתמקדת בהפשטת מערכת מורכבת
Mediator מתמקדת בתיאום בין אובייקטים, בעוד Facade מתמקדת בהפשטת מערכת מורכבת

תרגול

נסה ליישם את תבנית Facade במקרים הבאים:

  1. מערכת ניהול קבצים עם תמיכה בפורמטים שונים
  2. מערכת תקשורת עם פרוטוקולים שונים
  3. מערכת ניהול בסיסי נתונים עם תמיכה במנועים שונים
