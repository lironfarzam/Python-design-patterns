Bridge Pattern

חזרה לדף הבית

מטרת התבנית

תבנית Bridge מפרידה בין ממשק למימוש, ומאפשרת לשניהם להתפתח באופן עצמאי.

מתי להשתמש?

  • כאשר רוצים למנוע קשר קבוע בין ממשק למימוש
  • כאשר רוצים לאפשר לממשק ולמימוש להתפתח באופן עצמאי
  • כאשר רוצים להסתיר את המימוש מהלקוח
  • כאשר רוצים לשתף מימוש בין מספר ממשקים

דוגמה בסיסית 

class Implementation:
    def operation_implementation(self) -> str:
        return "מימוש בסיסי"

class ConcreteImplementationA(Implementation):
    def operation_implementation(self) -> str:
        return "מימוש A"

class ConcreteImplementationB(Implementation):
    def operation_implementation(self) -> str:
        return "מימוש B"

class Abstraction:
    def __init__(self, implementation: Implementation):
        self.implementation = implementation

    def operation(self) -> str:
        return f"Abstraction: {self.implementation.operation_implementation()}"

class ExtendedAbstraction(Abstraction):
    def operation(self) -> str:
        return f"ExtendedAbstraction: {self.implementation.operation_implementation()}"

# שימוש
def client_code(abstraction: Abstraction) -> None:
    print(abstraction.operation())

if __name__ == "__main__":
    implementation = ConcreteImplementationA()
    abstraction = Abstraction(implementation)
    client_code(abstraction)

    implementation = ConcreteImplementationB()
    abstraction = ExtendedAbstraction(implementation)
    client_code(abstraction)

יתרונות

  • מפרידה בין ממשק למימוש
  • מאפשרת לממשק ולמימוש להתפתח באופן עצמאי
  • מאפשרת שיתוף מימוש בין מספר ממשקים
  • מאפשרת החלפה קלה של מימוש

חסרונות

  • עלולה ליצור קוד מורכב
  • עלולה ליצור קוד קשה לתחזוקה
  • עלולה ליצור קוד קשה להבנה
  • עלולה ליצור קוד קשה לבדיקה

השוואה לתבניות אחרות

תבנית הבדל
Adapter מתמקדת בהתאמת ממשקים, בעוד Bridge מפרידה בין ממשק למימוש
Strategy מתמקדת בהחלפת אלגוריתמים, בעוד Bridge מתמקדת בהפרדת ממשק ומימוש

תרגול

נסה ליישם את תבנית Bridge במקרים הבאים:

  1. מערכת UI עם תמיכה בפלטפורמות שונות
  2. מערכת ניהול בסיסי נתונים עם תמיכה במנועים שונים
  3. מערכת עיבוד מסמכים עם פורמטים שונים
